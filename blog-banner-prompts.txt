HOW TO USE
================================================================================

Option A — automated (all 5 at once, plus features + bento = 17 images total):
  cd /mnt/d/Prabakaran/1.projects/ai-coloring-book
  npm run pregen:visuals -- --force

Option B — manual, one at a time in Google AI Studio:
  1. Open https://aistudio.google.com
  2. Select model: gemini-2.5-flash-image
  3. Set aspect ratio: 16:9
  4. Paste one of the 5 prompts above + the shared brand style appendix
  5. Generate, download as PNG
  6. Save to:  public/visuals/blog/<filename>.png
     (filename shown at the top of each banner section)

Option C — manual, one at a time in ChatGPT / DALL·E 3 / Midjourney:
  Same approach. DALL·E 3 handles 16:9 well; Midjourney use --ar 16:9 flag.
